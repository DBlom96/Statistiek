\question{1.m5}{
    De relatieve frequentie van de klasse $16 -< 20$ bedraagt $\dots$
}
\begin{enumerate}[label=(\alph*)]
    \item $0.05$
    \item $4$
    \item $16$
    \item $0.20$
\end{enumerate}
\answer{
    Bij $4$ van de $80$ busreizen is het aantal afzeggers in de klasse $16 -< 20$, oftewel de relatieve frequentie is $\frac{4}{80} = 0.05$.
    Het juiste antwoord is dus (a).
}
