\question{7.m3}{
    Het aantal ongevallen per maand op een kruispunt kan worden beschreven door een Poissonverdeling met $\mu=0,5$.
    De standaarddeviatie van het aantal ongelukken per jaar bedraagt daarom $\ldots$
}
\begin{enumerate}[label=(\alph*)]
    \item $6$.
    \item $\sqrt{6}$.
    \item $\frac{0,5}{\sqrt{6}}$.
    \item $0,5 \times \sqrt{12}$.
\end{enumerate}
\answer{
    Het aantal ongevallen per jaar kan in dat geval worden beschreven door een Poissonverdeling met $\mu = 12 \cdot 0,5 = 6$.
    De standaarddeviatie is dan gelijk aan $\sqrt{\mu} = \sqrt{6}$.
    Het juiste antwoord is dus (b).
}
